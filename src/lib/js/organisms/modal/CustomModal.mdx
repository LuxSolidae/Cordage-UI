#### CustomModal

### Props

<Props of={ CustomModal } />

<Playground>
  {() => {
    const [showSuccess, setShowSuccess] = React.useState(false);
    const [showError, setShowError] = React.useState(false);
    return (
      <ThemeContext.Provider theme={theme}>
        <Button variant='success' onClick={() => {
          setShowSuccess(true)
        }}>
          Click to open success modal
        </Button>
        <CustomModal
          header='Header of modal'
          isSuccess
          color='dark'
          show={showSuccess}
          onClose={() => {
            alert('Success modal closed');
            setShowSuccess(false);
          }}
        >
          We have sent an invitation with an expiration date of 3 Days. Please inform your users and have them check their emails. In case they don’t receive a message ask them to check their spam folder. This action make take up to 5 minutes.
        </CustomModal>
        <Button variant='success' onClick={() => {
          setShowError(true)
        }}>
          Click to open error modal
        </Button>
        <CustomModal
          header='Header of modal'
          isError
          color='dark'
          show={showError}
          onClose={() => {
            alert('Error modal closed');
            setShowError(false);
          }}
        >
          We have sent an invitation with an expiration date of 3 Days. Please inform your users and have them check their emails. In case they don’t receive a message ask them to check their spam folder. This action make take up to 5 minutes.
        </CustomModal>
      </ThemeContext.Provider>
    );
  }}
</Playground>

### Confirmation Modal

<Playground>
  {() => {
    const [showConfirmation, setShowConfirmation] = React.useState(false);
    return (
      <>
        <Button variant='primary' onClick={() => {
          setShowConfirmation(true);
        }}>
          Click to open Confirmation Modal
        </Button>
        <ConfirmationModal
          header='Confirm action'
          subheader='Are you sure you wish to proceed with this action?'
          btnCancel='Cancel'
          show={showConfirmation}
          onCancel={() => {
            alert('Cancel button clicked');
            setShowConfirmation(false);
          }}
          btnConfirm='Yes, confirm'
          onConfirm={() => {
            alert('Confirm button clicked');
            setShowConfirmation(false);
          }}
          onClose={() => {
            alert('Modal closed without action');
            setShowConfirmation(false);
          }}
        >
          The elimination of a user does not delete the activities the users have performed previously. There will be no implications in your organization’s Logs.
        </ConfirmationModal>
      </>
    );
  }}
</Playground>

### Generic Modal

This is a basic, empty modal. Buttons are optional, and if you need them, you can add a function too, just like a Confirmation Modal.

<Props of={GenericModal} />

<Playground>
  {() => {
    const [showGeneric, setShowGeneric] = React.useState(false);
    return (
      <>
        <Button variant='primary' onClick={() => {
          setShowGeneric(true);
        }}>
          Click to open Generic Modal
        </Button>
        <GenericModal
          header='Generic Modal'
          subheader='Generic Modal subheader.'
          show={showGeneric}
          onClose={() => {
            alert('Modal closed without action');
            setShowGeneric(false);
          }}
          primaryAction={{
            text: 'Accept',
            onClick: () => {
              alert('Accept clicked');
              setShowGeneric(false);
            },
          }}
          secondaryAction={{
            text: 'Decline',
            onClick: () => {
              alert('Decline clicked');
              setShowGeneric(false);
            },
          }}
        >
          Stuff...
        </GenericModal>
      </>
    );
  }}
</Playground>
